## ⁉️ Problems

### 🔹 Exercise 1 – Student Information
- 📄 Description: Write a program that inputs a student's **full name, Math score, and Literature score**.  
Calculate the **average score**, classify the student, and display the result.
- 💡 Idea: Compute average = (Math + Literature) / 2, then classify based on score ranges.
- 📂 Code: [Exercise_1.cpp](./Exercise_1.cpp)

---

### 🔹 Exercise 2 – Circle Calculation
- 📄 Description: Write a program that inputs the **center coordinates** and **radius** of a circle.  
Calculate the **area** and **circumference** of the circle.
- 💡 Idea: Use formulas: Area = πr², Circumference = 2πr.
- 📂 Code: [Exercise_2.cpp](./Exercise_2.cpp)

---

### 🔹 Exercise 3 – Distance Between Two Points
- 📄 Description: Write a program that inputs the coordinates of **two points in the Oxy plane**.  
Calculate the **distance between them** and display the result.
- 💡 Idea: Use distance formula √((x₂ - x₁)² + (y₂ - y₁)²).
- 📂 Code: [Exercise_3.cpp](./Exercise_3.cpp)

---

### 🔹 Exercise 4 – Array Class (cArray)
- 📄 Description: Implement a class `cArray` to represent a one-dimensional array of integers with the following features:
- Initialize an array of size *n* with **random integers** (constructor)
- Display the array
- Find the **largest negative number**
- Count occurrences of a given integer *x*
- Check if the array is in **descending order**
- Sort the array in **ascending order**
- 💡 Idea:
  - Generate random numbers using constructor
  - Traverse array to find max negative
  - Use loop to count occurrences
  - Check descending by comparing adjacent elements
  - Sort using basic sorting (e.g., selection sort)
- 📂 Code: [Exercise_4.cpp](./Exercise_4.cpp)

---

### 🔹 Exercise 5 – Student Class (cHocSinh)
- 📄 Description: Define a class `cHocSinh` with the following attributes:
- ID, full name, gender, year of birth, average score  

Write a program to:
- Input information for **two students**
- Determine which student has a **higher average score**
- Determine which student is **younger**
- 💡 Idea:
  - Store attributes in class
  - Compare average scores
  - Calculate age from birth year
- 📂 Code: [Exercise_5.cpp](./Exercise_5.cpp)

---

### 🔹 Exercise 6 – Complex Number Class
- 📄 Description: Implement a class to represent **complex numbers** (real and imaginary parts).

Provide the following functionalities:
- Initialize a complex number
- Modify real and imaginary parts
- Retrieve real and imaginary parts
- Input and output complex numbers
- Perform operations: **addition, subtraction, multiplication, division**

For two complex numbers A(a₁, a₂) and B(b₁, b₂):
- A + B = (a₁ + b₁, a₂ + b₂)  
- A − B = (a₁ − b₁, a₂ − b₂)  
- A × B = (a₁b₁ − a₂b₂, a₁b₂ + a₂b₁)  
- A ÷ B = ((a₁b₁ + a₂b₂) / (b₁² + b₂²), (b₁a₂ − a₁b₂) / (b₁² + b₂²))

Write a program to demonstrate all functionalities of the class.
- 💡 Idea:
  - Store real & imaginary parts
  - Apply formulas for +, −, ×, ÷
  - Use operator overloading (if learned)
- 📂 Code: [Exercise_6.cpp](./Exercise_6.cpp)
